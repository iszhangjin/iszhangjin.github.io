# 前言

    做出正确的决定，制定一个长期计划，使自己获得财务保障，甚至创造出一份财富，以此来改善自己的生活质量。

# 问题
- 如何消除对金钱和财富的偏见？
- 怎样才能避免消费债务？
- 如何在不丧失生活乐趣的情况下正确省钱，让你的收入大幅提高？

# 金钱可以带来什么？
    金钱应该给予我们安全感、自由感。金钱应该为我们提供这样一种可能性，一种我们可以过上自己想要的生活的可能性，金钱可以使人更接近于幸福的生活。
